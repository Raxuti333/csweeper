## Building

### Dependencies

[GLFW](https://www.glfw.org/)

[zlib](https://zlib.net/)

### Compiling

install python depencies by running: 
``` 
    pip install -r requirements.txt
```

change variables CC and CC_WIN in build.py to use your prefered C compiler
```
    python build.py
```
